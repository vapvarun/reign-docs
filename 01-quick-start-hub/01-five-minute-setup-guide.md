# 5-Minute Setup Guide for Reign Theme

## Quick Setup Overview

Get your Reign theme up and running in just 5 minutes with this streamlined setup process. This guide covers the essential steps to launch your community site quickly.

## Prerequisites

- WordPress 5.0 or higher installed
- PHP 7.4 or higher
- MySQL 5.6 or higher
- BuddyPress plugin (optional but recommended)

## Step 1: Install Reign Theme (1 minute)

### Via WordPress Admin
1. Navigate to **Appearance > Themes > Add New**
2. Click **Upload Theme**
3. Choose the `reign-theme.zip` file
4. Click **Install Now**
5. Click **Activate**

### Via FTP (Alternative)
1. Extract `reign-theme.zip` on your computer
2. Upload the `reign-theme` folder to `/wp-content/themes/`
3. Go to **Appearance > Themes** in WordPress admin
4. Activate Reign theme

## Step 2: Activate Your License (30 seconds)

1. Go to **Appearance > Reign Settings**
2. Click on the **License** tab
3. Enter your purchase code
4. Click **Activate License**

**Benefits of activation:**
- Automatic theme updates
- Access to premium support
- Unlock all theme features
- Access to demo content

## Step 3: Install Required Plugins (1 minute)

After activation, you'll see a notice to install recommended plugins:

1. Click **Begin installing plugins**
2. Select all plugins (or choose specific ones):
   - **Kirki Framework** (Required for customizer)
   - **BuddyPress** (For community features)
   - **WooCommerce** (For e-commerce)
   - **Elementor** (For page building)
3. Click **Install** from the dropdown
4. Click **Apply**
5. Return to dashboard and **Activate** plugins

## Step 4: Import Demo Content (1.5 minutes)

1. Go to **Appearance > Reign Demo Import**
2. Choose a demo that matches your needs:
   - **Community Demo** - Social network layout
   - **Learning Demo** - Educational platform
   - **Marketplace Demo** - Multi-vendor setup
   - **Membership Demo** - Subscription site
3. Click **Import Demo**
4. Wait for import completion

**What gets imported:**
- Sample pages and posts
- Menu structures
- Widget configurations
- Customizer settings
- Sample users (for community demos)

## Step 5: Essential Configuration (1 minute)

### Set Your Logo
1. Go to **Appearance > Customize > General > Site Identity**
2. Upload your logo (recommended: 200x50px)
3. Upload mobile logo (optional)
4. Set site title and tagline

### Configure Primary Menu
1. Go to **Appearance > Menus**
2. Select **Primary Menu**
3. Add your main pages
4. Set display location to **Primary Menu**
5. Save menu

### Set Homepage
1. Go to **Settings > Reading**
2. Choose **A static page**
3. Set Homepage (imported from demo)
4. Set Posts page (if using blog)
5. Save changes

## Troubleshooting Quick Setup

### Theme Not Activating
- Check PHP version (must be 7.4+)
- Increase memory limit to 256M
- Ensure theme files uploaded completely

### Demo Import Failing
- Increase PHP max_execution_time to 300
- Increase max_input_vars to 3000
- Try importing in smaller chunks

### Plugins Not Installing
- Check file permissions
- Verify WordPress can write to plugins directory
- Install plugins manually if needed
