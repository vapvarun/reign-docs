## Database Integration

These documentation files are designed to be imported into WordPress as 'docs' post type with 'doc_category' taxonomy set to 'wb-reign-theme' and appropriate subcategories.

### Import Process:
1. Each .md file can be converted to WordPress post
2. Set post_type = 'docs'
3. Assign to doc_category = 'wb-reign-theme'
4. Add appropriate subcategory based on folder structure
5. Maintain hierarchical organization
