## Course Creation & Management

### How do I create my first course?

**Step-by-step for beginners:**

1. **Plan your course:**
   ```
   - What will students learn?
   - How long will it take?
   - What materials do you need?
   ```

2. **Create the course:**
   ```
   Courses ‚Üí Add New
   Add title, description, featured image
   Click Publish
   ```

3. **Add lessons:**
   ```
   In course editor ‚Üí Add Lesson
   Upload videos, add text, attachments
   Arrange in logical order
   ```

4. **Add a quiz (optional):**
   ```
   Add Quiz ‚Üí Create questions
   Set passing grade
   Associate with lesson
   ```

5. **Set pricing:**
   ```
   Course Settings ‚Üí Access Plans
   Choose: Free, One-time, Subscription
   Set price and options
   ```

6. **Test everything:**
   ```
   Enroll as test student
   Go through entire course
   Fix any issues
   ```
